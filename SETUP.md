# Quick Setup - RAG Demo

## ✅ What's Ready

All 5 demos are configured for Windows and ready to run!

## 🚀 Run Demos

```bash
cd 04_Hands_On_Code

# Demo 1 - Basic RAG
python demo_1_basic_rag.py

# Demo 2 - Chunking strategies
python demo_2_chunking_strategies.py

# Demo 3 - Vector DB & Retrieval
python demo_3_vector_storage_and_retrieval.py

# Demo 4 - Evaluation Metrics
python demo_4_evaluation_metrics.py

# Demo 5 - Production RAG
python demo_5_complete_rag_system.py
```

## 📝 Setup Instructions

**Before starting:**
1. Install Python 3.8+
2. Get Gemini API key from https://makersuite.google.com/app/apikey
3. Create `.env` file with: `GOOGLE_API_KEY=their-key`

**During session:** Run demos and explain!
