# RAG Demo for BCA Students - Complete Package

## 📦 What's Included

This package contains everything you need for a 90-minute interactive RAG workshop for 150 BCA students.

### 🎯 Teaching Philosophy
- **Start Simple**: Paper-based intuition before technical concepts
- **Progressive Complexity**: Kid → Intermediate → Pro levels
- **Interactive**: Quiz after every major concept
- **Hands-On**: Live coding with Gemini API
- **Reward-Based**: Goodies for participation

### ⏱️ 90-Minute Timeline

| Time | Activity | Duration |
|------|----------|----------|
| 0-10 min | Paper Exercise (Ice-breaker) | 10 min |
| 10-25 min | RAG Basics + Quiz 1 | 15 min |
| 25-45 min | Chunking Deep Dive + Demo 1-2 + Quiz 2 | 20 min |
| 45-65 min | Vector DB & Retrieval + Demo 3 + Quiz 3 | 20 min |
| 65-85 min | Advanced Topics + Demo 4-5 + Quiz 4 | 20 min |
| 85-90 min | Wrap-up & Goodies Distribution | 5 min |

### 🎁 Prize Strategy

- **Quiz Winners**: Top 3 teams per quiz (12 prizes total)
- **Best Question**: Most insightful question (1 prize)
- **Completion Award**: All teams who complete the challenge project

### 🚀 Quick Start for Instructor

1. Print `01_Paper_Exercise/Sheet_A_Unorganized.pdf` (150 copies, back-to-back with Sheet B)
2. Open `02_Slides/RAG_Demo_Slides.html` in browser
3. Set up quiz platform (Google Forms or Kahoot links in quizzes folder)
4. Test run `demo_1_basic_rag.py` to verify Gemini API key
5. Review `INSTRUCTOR_GUIDE.md` for delivery tips
